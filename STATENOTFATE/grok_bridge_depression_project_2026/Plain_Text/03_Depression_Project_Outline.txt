Depression Project Outline
Master outline | April 2026
I. Project thesis
Depression is treated as a systems failure affecting energy, perception, reward prediction, behavior, and function.
The project aims to restore usable function through repeated structural corrections rather than motivational pressure.
II. Design principles
Function over appearance
Repetition over intensity
Anchors over inspiration
Low-friction execution over ideal execution
State-based planning over one universal standard
