What changed in this rebuild
The front end is now treated as a separate operating layer rather than something buried inside the long manual.
The project logic is written around systems failure, function loss, anchor stacking, and low-energy usability.
The newer direction that treats hope as a measurable prediction signal is given its own dedicated front-end file.
The bundle is built to be portable: easy to move, easy to skim, and easier to edit later.
