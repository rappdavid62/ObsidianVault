# ChatGPT Export -> Grok

If you want broader chat history, not just the cleaned project docs, export your ChatGPT data and then upload only the relevant pieces into Grok.

Recommended use:
- export ChatGPT data
- pull out the relevant project conversations or summaries
- convert those into smaller markdown or text files
- upload those to Grok Collections or attach them in a dedicated Grok project chat

Do not dump a giant raw export into Grok first.
Curate it.
The better approach is:
1. cleaned bundle
2. project brief
3. selected chat summaries
4. only then larger exports if needed
