# Start Here

This folder is the working entry point for David's AI / Obsidian / project system.

## What to do first

1. Open `AI_SYSTEM/CURRENT_STATUS.md`.
2. Update anything that is wrong today.
3. Open `CAPTURE/RAW_INBOX.md`.
4. Dump any messy thoughts there.
5. Use AI later to clean raw material into the right file.

## Main Rule

Do not over-organize while tired.

Capture first. Sort later.

## Folder Map

- `AI_SYSTEM/` — durable AI context and operating rules
- `PROJECTS/` — active project notes
- `JOB_SEARCH/` — job search strategy, trackers, resumes, applications
- `LIFE_ADMIN/` — money, benefits, documents, calls
- `CAPTURE/` — raw thoughts, voice dumps, unsorted material
- `PROMPTS/` — reusable prompts and AI commands

## Daily Minimum

Open `CURRENT_STATUS.md` and update one line.

That is enough to keep the system alive.
