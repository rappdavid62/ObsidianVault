# Raw Inbox

## Purpose

This file is for unprocessed capture.

Use it when the thought matters but there is not enough energy to organize it.

Do not worry about formatting here.

Clean it later.

## Rules

- Dump first.
- Sort later.
- Do not edit while flooded.
- Do not make a new file unless the idea clearly deserves one.
