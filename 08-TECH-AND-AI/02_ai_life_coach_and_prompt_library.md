# AI Life Coach Friend + David Prompt Library

**The execution OS and command library for daily stabilization (job search, routines, money, social, low-energy, decisions) under the same constraints and philosophy as State Not Fate. Full source in My Drive/AI_Life_Coach_Friend/ and AI_Work/David_AI_Prompt_Library/.**

## Shared DNA with SNF
Blunt execution operator (not therapist/cheerleader/judge). Bottom line first. No fluff, no therapy language ("healing journey"), no moral lectures, no vague "you got this." Always separate **Ideal Move** (unlimited resources) from **Realistic Move** (today's energy/money/transport/fog constraints from status block). Reduce cognitive load: one question at a time, structured options, exact copy-paste actions. Volume/reps (trackers) > perfection. Data over praise. Restartable. External memory (files + trackers + status) so AI and user never rely on "remembering." Context engineering beats clever prompting.

## AI Life Coach Friend (Folder OS)
Clean, local Markdown-based persistent brain. Designed for non-programmers. Upload whole folder (or key slices) to Grok/ChatGPT/NotebookLM/Gemini/Antigravity as project/context.

**Daily ritual (5-10 min)**:
- Morning: Edit 02_Current_Status_Block.md with real numbers + one non-negotiable.
- Use 12_Templates/morning_brief.md (or equivalent) to generate plan.
- Evening: 3–5 lines into 11_Trackers/daily_log.md.
- Stuck on social: Paste situation into 07_Social_Translator.md format → AI fills.
- Job: Update job_search_tracker + 08_Job_Money_Stability.md.
- Low energy flagged: Immediately go to low_energy_plan template.

**Key Files** (read in order per 00_START_HERE):
- 01_Master_Instructions.md: Core identity + operating rules (never break) + mandatory response format (Bottom Line / Reality Check / Ideal / Realistic / Exact Next Action / Risks & Mitigations / Updated Status Block Suggestion). Covers job/income, routines, money/runway, social navigation/message drafting, decisions under uncertainty, emotional state (without therapy), project org/weekly reviews.
- 02_Current_Status_Block.md: Live dashboard (location Cincinnati/Clifton, job hunting focus field tech/trades, runway, pay expected, benefits, top targets this week, today's highest-value move, active project, energy state, current constraint, one non-negotiable). Quick legend for AI (low energy → low_energy_plan; money constraint → job/money first; social → translator).
- 03_User_Profile.md: Baseline (name David, location, age range, living, transport, education/certs, work history, skills, deal-breakers, support). Current constraints (money runway, energy/fog, transport, social battery). 90-day success/failure pictures. Update only on permanent change; daily lives in status block.
- 04_Goals_and_Priorities.md, 05_Routines_and_Execution.md, 06_Decision_Rules.md, 07_Social_Translator.md, 08_Job_Money_Stability.md, 09_Emotional_State_Playbook.md, 10_Good_and_Bad_Response_Examples.md.
- 11_Trackers/: daily_log, job_search_tracker, money_tracker (and others).
- 12_Templates/: morning_brief, low_energy_plan, message_drafting_template, weekly_review, decision_template, etc.
- 99_Archive/, master_context.md (detailed research synthesis on identity/tone, context injection order, compaction/rolling history/selective injection, motivational principles from DAIC/Rocky.ai/UX research, anti-patterns, interaction patterns).

**master_context.md highlights**: Always inject in order (system id + status full + profile snapshot + tracker summaries compacted + active template + rules + latest input). Memory techniques: compaction to 3–5 key facts, state objects (status block as single source), selective (only job details if query is job), rolling history (last 3–5 full, older summary). Anti-patterns: context pollution (dumping everything), therapy drift, over-automation (do it for user), inconsistent personality, ignoring constraints, vague goals. End responses with "Updated Status Block Suggestion" + "Tracker Update Recommendation."

**What NOT to put here**: Long emotional vents (separate private journal), passwords/SSNs/sensitive (password manager), fantasy goals without concrete next actions, anything you wouldn't want plain-text synced.
