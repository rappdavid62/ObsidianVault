## Core Principles (Stand-out Revelatory Connective Pieces)
- **"State is information. Fate is a story we stop telling."** (ANTI-GRAVITY 2.0). Current energy/mood/capacity/symptoms = data, never identity or permanent. Neuroplasticity via small repeated self-compassionate actions.
- Depression = multi-system assault (bio/psycho/social/behavioral/existential) that damages **startup/initiation**, reward registration, self-trust, future horizon. Not fixed by "just exercise" or generic advice (those assume the capacities it breaks).
- **Proof-based hope**, not hype: Level 1 action still possible → ... → Level 5 stable life supports wider future. First win = traction, not transformation. Floor Wins legitimate (no performance requirement).
- **Restart fidelity over compliance**. Misses happen; shame expands them. Design for re-entry: no catch-up debt, no shame, "I restart without punishment." Systems survive bad days via low friction, visible cues, short action→consequence loops.
- **Ideal vs Realistic** (always separate in AI coach responses and plans). Realistic survives contact with actual constraints (low money, fog, inconsistent energy, work schedule, transport).
- **Energy sovereignty**: 4 tiers (High / Medium / Low / Collapse) → MVD (Minimum Viable Day) + adaptive anchors/Polaris UI. High = expansion; Collapse = safety + regulation only.
- **Anchors + Mantras + Meditation + Circadian + Graded movement** as load-bearing infrastructure. Mantras = deliberate counter-scripts installed by repetition aloud (examples in philosophy note). Meditation creates gap between depressive thought and next action.
- **Matched first moves** via intake (burden, friction, capacity, trust, restartability). Front-end trust before full complexity.
- **Stabilization before expansion**. Floor first (reduce chaos, build anchors, repeat wins, body/rhythm/work), *then* social/creative/relationship/identity/life-direction (5-year vision: stable partnership + family in Cincinnati).
- **Workability under real constraint**: cheap, minimal equipment, fits employment, brain-fog friendly, no life-coach dependency. "One small action still counts." "Action comes before motivation."
- **Human tone** (blunt but not contemptuous; clear not simplistic; respectful of low functioning; free of empty inspiration; specific and usable today). Shared across SNF PWA, Anti-Gravity Engine, AI Life Coach Friend.
- **External memory & context engineering** (AI Life Coach + trackers + status blocks + Obsidian/GitHub): "Context engineering beats clever prompting." Re-inject status, profile, trackers, rules every time. No memory games for the model or user.
- **AI as execution operator / Anti-Gravity Engine**: Bottom Line first. Ideal/Realistic/Exact Next Action. Update status/tracker at end. Floor Win framing on low days. Never therapy drift or yes-man. Reference actual PWA UI elements (Polaris tab, anchors checklist, Log Floor Win 🔥, toasts).
- **Fantasy boundary**: Creative/roleplay (BE source materials, GF MODE ON) strictly compartmentalized from recovery work.
- **Professional care as primary**: SNF is adjunctive self-management OS. Med adherence (when prescribed), therapy, crisis (988), clinical review invited. PHQ-9 uses *exact* clinical language (never paraphrase).
- **Lived shaping**: User's history (breakdown, homelessness, isolation, low money, fog, past work as mover/driver/packer, PRS cert APS.006470, sober since 2019-11-01, running, voice acting, social reintegration) makes trust fragile and demands low-friction, believable, restartable, non-moralizing design. (From bundles' "user's own autobiographical and functional testimonial" + profile/status.)
