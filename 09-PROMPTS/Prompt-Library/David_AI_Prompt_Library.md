# Council Strategy Command

Use this for complex, strategic, job, money, social, health, legal-ish, public-facing, or high-consequence decisions.

```text
/council

Apply the Council Method Source Document v2 as the operating method. Do not summarize it. Compress or expand the format according to task complexity.

Task:
[describe the task]

Context:
[what is happening]

Goal:
[what I want to decide, produce, avoid, or execute]

Constraints:
- Money:
- Time/deadline:
- Energy/motivation:
- Social/reputation risk:
- Technical limits:

Available sources:
[pasted text, screenshots, files, links, job posts, notes, or say none]

What I already tried:
[briefly say what has happened]

What I am worried about:
[briefly say the risk]

Output wanted:
[decision / plan / draft / critique / research brief / checklist / workflow / message]

Rules:
- Start with the bottom line.
- State what sources/tools are actually available.
- Separate evidence, inference, speculation, and strategic judgment.
- Ask a clarifying question only if truly blocking.
- End with the practical next move.
```

# Social Calibration Command

Use this for texts, DMs, comments, dating, replies, interviews, awkward situations, and people-reading.

```text
Translate this social situation plainly.

Situation:
[paste screenshot/text/context]

My goal:
[what I want]

Relationship/context:
[who they are to me]

Before drafting, give:
1. Likely meaning
2. How I am probably being perceived
3. Social risk
4. Best response strategy
5. Send/no-send verdict if a message is involved
6. Target
7. Objective
8. Missing facts
9. Timing

Then give:
- Best response
- Safer version
- Bolder version
- What not to say

Tone:
Plain, socially literate, not needy, not robotic, not overexplained. Preserve intelligence and controlled weirdness when appropriate.
```
