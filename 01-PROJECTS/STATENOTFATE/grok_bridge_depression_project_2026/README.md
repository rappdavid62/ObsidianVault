# Grok Bridge Bundle - Depression Project

This folder is a Grok-ready handoff pack built from the current core Depression Project documents.

## What is inside
- Raw_Docs: original working documents
- Plain_Text: text exports for easier AI ingestion
- Prompts: ready-to-use Grok prompts
- Bridge: instructions for moving this project between ChatGPT and Grok
- Manifest: upload order and machine-readable maps

## Fastest use
1. Upload 00_START_HERE.md first.
2. Upload Prompts/GROK_SYSTEM_PROMPT.md or paste it into Grok.
3. Upload or attach the most important docs from Raw_Docs.
4. If Grok supports Collections, upload this whole set there.
5. If you want a recurring bridge, store this folder in a dedicated Google Drive folder.
