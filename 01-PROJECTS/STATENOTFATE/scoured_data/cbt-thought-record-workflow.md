# CBT Thought Record Workflow

## Source Link
[https://www.beckinstitute.org](https://www.beckinstitute.org)

## Curated Summary
Cognitive Behavioral Therapy (CBT) thought records help patients identify cognitive distortions (e.g., catastrophizing, black-and-white thinking) and actively correct them.

## Suggested Exercises / Protocols
1. Write down the situation and the negative emotion (0-100%).
2. Extract the automatic hot thought.
3. Write evidence that supports the thought.
4. Write evidence that contradicts the thought.
5. Formulate a balanced, realistic alternative thought.
6. Re-rate your emotion.
