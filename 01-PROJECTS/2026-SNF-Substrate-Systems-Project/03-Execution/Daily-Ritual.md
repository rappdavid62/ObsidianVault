# Daily Ritual for 2026 SNF Substrate Systems Project

**Goal**: Protect the floor + generate at least one visible substrate-shaping proof every day using the Library.

## Morning (Low friction, ~5-10 min)
1. Update status block (energy, runway, one non-negotiable).
2. Emit or recall: substrate-reminders + mvd-anchors + sobriety-anchors + circadian-anchors.
3. Choose 1 tiny action for today that will be made external/visible.
4. Say or write one core reminder phrase: "This is proof that effort can still shape the substrate."

## During the Day
- When resistance or "what's the point" hits: Use substrate-reminders skill (or the phrases above).
- Any action completed: Immediately register via snf-proof-registration (log in 04-Tracking/Proof-Log.md + main trackers).

## Evening
1. Review what actually happened.
2. Register at least one floor win / substrate proof in the log.
3. Quick systems check: Did today's action touch any load-bearing system?
4. Log in main daily note or this project's 04-Tracking.

**Always use the full Library as source of truth. Emit custom packs when stuck (e.g. substrate-reminders + systems-audit + 5-year-vision-alignment).**

**No day with effort is unregistered. Small consistent proof reshapes the substrate.**
