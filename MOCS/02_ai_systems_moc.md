# AI Systems & Execution MOC

See [[00_grok_organization_log]] for the session that assembled and linked these piles.

**Hub for the AI coaching, prompt, and execution layers that operationalize State Not Fate principles in daily life (job, routines, money, social, low energy, decisions).**

These systems share DNA with the SNF project: blunt execution operator, bottom-line first, no fluff/therapy/moralizing, **Ideal vs Realistic Move** given real constraints (energy, money, fog, transport), external memory via status/tracker files, proof/volume over perfection, one non-negotiable, restartable, low cognitive load.

## How They Tie to State Not Fate
- Same principles: State ≠ Fate; proof/anchors; low friction; restart without shame; ideal/realistic split; energy awareness; external memory (status blocks + trackers instead of relying on AI "remembering"); blunt tone; stabilization (job/routines/money as floor for the 5-year vision).
- AI Life Coach Friend + Prompt Library = execution tooling for the "stabilization before expansion" phase (job search as high-value move for income stability + identity as PRS; daily anchors via low_energy_plan when fog hits).
- Anti-Gravity Engine (PWA-specific) + Life Coach Friend + Prompt Library = layered AI support. Use Life Coach/Prompts for broader life/job; switch to Anti-Gravity protocols when in the PWA (Polaris, Floor Win, exact UI).
- Past conversations (Grok sessions + ChatGPT exports synthesized into the comprehensive bundles) were the forge: recurring themes → master prompts → PWA features → this knowledge org task.
